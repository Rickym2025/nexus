# NexusAI Mirror Documentation

Questo archivio contiene i file statici della landing page di **NexusAI**, pronti per essere distribuiti su CDN globali (Vercel).

## Struttura del Repository
- `/index.html` : Landing Page principale del servizio.
- `/config.html` : Pagina di personalizzazione post-acquisto per i clienti.
- `/public/` : Cartella contenente gli asset della demo proxy.
- `/public/index.html` : Interfaccia del generatore di demo (visibile su `demo.rmstudio.app`).
- `/public/success.html` : Pagina di atterraggio post-pagamento Stripe.
